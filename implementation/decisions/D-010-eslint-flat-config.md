# D-010: ESLint flat config (`eslint.config.mjs`)

## Decisión

Usar ESLint flat config (`eslint.config.mjs`) en lugar de `.eslintrc.json`.

## Contexto

Angular 21 genera flat config por defecto; esquema más moderno y oficial.

## Alternativas descartadas

`.eslintrc.json` (legado, deprecado en ESLint 9+)
