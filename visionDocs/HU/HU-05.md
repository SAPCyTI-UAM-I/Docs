# Editar plan trimestral

## 1. Tarjeta
**ID:** HU-05

**Título:** Editar plan trimestral

**Como:** coordinador del programa

**Quiero:** editar el plan trimestral detallado

**Para:** definir horarios, profesores, grupos y cupos de las materias derivadas del plan anual

## 2. Conversación
* El coordinador inicia sesión en el sistema.
* En el menú lateral selecciona la opción **“Editar plan trimestral”** dentro del módulo **Proceso Planeación Trimestral**.
* El sistema muestra una pantalla donde se debe seleccionar el trimestre a editar.
* Una vez seleccionado el trimestre y presionado **“Continuar”**:

  * El sistema muestra las materias generadas con base en el número de grupos definidos en el plan anual.
  * Se visualiza una tabla con:

    * Clave
    * Materia
    * Grupo
    * Opción **Editar**
* Desde esta pantalla el coordinador puede:

  * Editar los detalles de una materia específica.
  * Modificar el plan (agregar o quitar materias respecto al plan anual).
  * Visualizar el plan completo actualizado.

### Edición de una materia

Al seleccionar **“Editar”**, el sistema muestra una ventana donde se pueden modificar los siguientes campos:

* **Horario**

  * Selección de días (Lunes a Viernes).
  * Hora de inicio.
  * Hora de fin.
* **Profesor**

  * Selección desde una lista desplegable de profesores.
* **Grupo**

  * Campo editable.
* **Cupo**

  * Campo editable (puede modificarse respecto al plan anual).

### Caso especial: Proyectos de Investigación

* Se puede abrir:

  * Para un alumno específico.
  * Sin alumno asignado.
* El sistema muestra:

  * Checkbox para indicar si es para un solo alumno.
  * Combo-box con la lista de alumnos disponibles.

### Visualización del plan

* El coordinador puede seleccionar la opción para **ver el plan completo**.
* El sistema muestra una tabla consolidada con:

  * Clave
  * Materia
  * Grupo
  * Cupo
  * Profesor asignado
  * Número de empleado (cuando aplique)


## 3. Criterios de Aceptación
```gherkin
Feature: Edición de Plan Trimestral

  Scenario: Selección de trimestre a editar
    Given que el coordinador ha iniciado sesión
    When selecciona la opción "Editar plan trimestral"
    And elige un trimestre válido
    And presiona "Continuar"
    Then el sistema muestra la lista de materias asociadas al trimestre

  Scenario: Visualización de materias generadas desde el plan anual
    Given que el coordinador seleccionó un trimestre
    Then el sistema muestra las materias de acuerdo al número de grupos definidos en el plan anual
    And presenta la opción "Editar" para cada materia

  Scenario: Edición de detalles de una materia
    Given que el coordinador visualiza la lista de materias
    When selecciona la opción "Editar" en una materia
    Then el sistema muestra la ventana de edición
    And permite modificar horario
    And permite seleccionar profesor
    And permite editar grupo
    And permite actualizar cupo

  Scenario: Configuración de horario
    Given que el coordinador se encuentra editando una materia
    When selecciona días y define hora de inicio y fin válidas
    Then el sistema guarda correctamente el horario

  Scenario: Asignación de profesor
    Given que el coordinador se encuentra editando una materia
    When selecciona un profesor de la lista
    Then el sistema registra la asignación correctamente

  Scenario: Configuración de proyecto de investigación para alumno específico
    Given que la materia es un proyecto de investigación
    When el coordinador selecciona que es para un solo alumno
    And elige un alumno del combo-box
    Then el sistema guarda la materia asociada al alumno seleccionado

  Scenario: Visualización del plan completo
    Given que el coordinador está editando el plan trimestral
    When selecciona la opción para ver el plan completo
    Then el sistema muestra el listado consolidado del trimestre
    And despliega clave, materia, grupo, cupo y profesor asignado

  Scenario: Validación de datos inválidos
    Given que el coordinador edita una materia
    When ingresa datos inválidos en horario, grupo o cupo
    Then el sistema muestra un mensaje de error
    And no permite guardar hasta corregir la información
```
