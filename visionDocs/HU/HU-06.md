# Iniciar proceso de inscripción

## Tarjeta

**ID:** HU-06

**Nombre:** Inicio del proceso de inscripción trimestral y carga de horarios

**Como** Coordinador del Posgrado

**Quiero** seleccionar un trimestre y cargar el archivo CSV de horarios y sorteos

**Para** habilitar el sistema de inscripción y que los alumnos puedan visualizar las materias disponibles

## Conversación

* El coordinador selecciona el trimestre (ej. 16I) para "abrir" la inscripción.
* Debe cargar un archivo CSV con los horarios específicos y fechas de sorteo.
* El sistema valida el archivo al dar clic en "Aceptar".
* Si el archivo es correcto, los alumnos verán las materias en su portal.
* Si el CSV tiene formato incorrecto, el sistema arroja un error y no inicia el proceso.

## Criterios de Aceptación

```gherkin
Feature: Iniciar proceso de inscripción

  Scenario: Selección del trimestre para inscripción
    Given que el coordinador ha navegado a la opción "Iniciar proceso de inscripción"
    When visualiza el menú desplegable "Trimestre"
    And selecciona un periodo válido (ej. 16I)
    Then el sistema permite continuar con la carga del archivo

  Scenario: Carga exitosa del archivo CSV de horarios
    Given que el coordinador ha seleccionado un trimestre
    When presiona el botón "Elegir archivos"
    And selecciona un archivo con extensión ".csv" que contiene horarios y fechas de sorteo
    And presiona el botón "CARGAR"
    Then el sistema procesa la información exitosamente
    And muestra el nombre del archivo seleccionado junto al botón

  Scenario: Validación de formato de archivo incorrecto
    Given que el coordinador intenta subir la información de horarios
    When selecciona un archivo que no tiene extensión ".csv" (ej. .pdf o .docx)
    And presiona "CARGAR"
    Then el sistema muestra un mensaje de error
    And no permite iniciar el proceso de inscripción

  Scenario: Activación del proceso para la comunidad estudiantil
    Given que el archivo CSV ha sido cargado y validado correctamente
    When el coordinador presiona el botón "Aceptar"
    Then el sistema cambia el estado del trimestre a "En Inscripción"
    And los alumnos pueden visualizar las materias y horarios en sus respectivos portales

  Scenario: Intento de inicio sin archivo seleccionado
    Given que el coordinador ha seleccionado un trimestre
    But no ha elegido ningún archivo CSV
    When presiona el botón "Aceptar"
    Then el sistema muestra una alerta indicando que la carga del CSV es obligatoria

  Scenario: Persistencia de la planeación trimestral
    Given que se inicia el proceso de inscripción
    Then el sistema debe asegurar que las materias mostradas correspondan exactamente a lo definido en la "Planeación Trimestral" previa
```
