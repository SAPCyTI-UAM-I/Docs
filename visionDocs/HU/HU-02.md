# Recuperación de contraseña

## 1. Tarjeta
**ID:** HU-02

**Título:** Recuperación de contraseña

**Como:** usuario registrado del sistema SAPCyTI

**Quiero:** recuperar mi contraseña cuando la haya olvidado

**Para:** poder restablecer mi acceso al sistema



## 2. Conversación
* Desde la pantalla de ingreso (HU-01), el usuario puede seleccionar la opción **“¿Olvidaste tu contraseña?”**.
* El sistema redirige a la pantalla de **Recupera Contraseña**.
* En esta pantalla:

  * Se muestra el campo **Email**.
  * Se muestra el botón **Enviar Correo**.
* El usuario debe capturar el correo electrónico con el que está registrado en el sistema.
* Al presionar **Enviar Correo**:

  * El sistema valida que el correo tenga formato correcto.
  * Si el correo está registrado:

    * El sistema envía un correo electrónico con instrucciones para recuperar o restablecer la contraseña.
    * Muestra un mensaje de confirmación.
  * Si el correo no está registrado:

    * El sistema muestra un mensaje indicando que el correo no existe en el sistema.
* El usuario puede regresar a la pantalla de inicio mediante la opción **Inicio**.



##  Criterios de Aceptación (Gherkin)

```gherkin
Feature: Recuperación de contraseña en SAPCyTI

  Scenario: Acceso a la pantalla de recuperación de contraseña
    Given que el usuario se encuentra en la pantalla de login
    When selecciona la opción "¿Olvidaste tu contraseña?"
    Then el sistema muestra la pantalla "Recupera Contraseña"
    And presenta un campo para capturar el email
    And muestra el botón "Enviar Correo"

  Scenario: Envío exitoso de correo de recuperación
    Given que el usuario se encuentra en la pantalla "Recupera Contraseña"
    When ingresa un correo electrónico registrado con formato válido
    And presiona el botón "Enviar Correo"
    Then el sistema valida el correo
    And envía un correo con instrucciones para recuperar la contraseña
    And muestra un mensaje indicando que el correo fue enviado exitosamente

  Scenario: Correo no registrado
    Given que el usuario se encuentra en la pantalla "Recupera Contraseña"
    When ingresa un correo electrónico no registrado
    And presiona el botón "Enviar Correo"
    Then el sistema muestra un mensaje indicando que el correo no existe en el sistema
    And no envía ningún correo

  Scenario: Formato de correo inválido
    Given que el usuario se encuentra en la pantalla "Recupera Contraseña"
    When ingresa un correo electrónico con formato inválido
    And presiona el botón "Enviar Correo"
    Then el sistema muestra un mensaje indicando que el formato del correo es incorrecto
    And no procesa la solicitud

  Scenario: Regresar a la pantalla de inicio
    Given que el usuario se encuentra en la pantalla "Recupera Contraseña"
    When selecciona la opción "Inicio"
    Then el sistema redirige a la pantalla de login
```
