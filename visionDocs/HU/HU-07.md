# Elegir materias (UEA)

## Tarjeta

**ID:** HU-07

**Nombre:** Selección de Unidades de Enseñanza Aprendizaje (UEA) por el alumno

**Como** Alumno del posgrado

**Quiero** acceder al módulo de inscripción para seleccionar las materias que cursaré en el trimestre

**Para** registrar mi carga académica y avanzar en mi programa de estudios

## Conversación

* El alumno entra al sistema y selecciona la opción "Inscripción" en el menú lateral.
* El sistema muestra primero los datos personales (Nombre, Nivel, Carrera, etc.) para su verificación.
* Se despliega una lista de materias (UEA). Algunas pueden aparecer pre-seleccionadas automáticamente según el plan.
* El alumno puede marcar o desmarcar las UEA según considere necesario.
* Se pueden revisar los detalles de cada materia: clave, nombre, grupo, horarios y profesor.
* Al finalizar, el alumno guarda su elección para continuar con el proceso.

## Criterios de Aceptación

```gherkin
Feature: Elegir materias (UEA)

  Scenario: Acceso al módulo de inscripción
    Given que el alumno ha iniciado sesión en el sistema SAPCyTI
    And el proceso de inscripción ha sido iniciado por el coordinador
    When selecciona la opción "Inscripción" en el menú "ALUMNO"
    Then el sistema muestra la pantalla de inscripción con los datos del alumno

  Scenario: Verificación de información personal
    Given que el alumno se encuentra en la pantalla de inscripción
    Then el sistema despliega el nombre, nivel (ej. MAESTRÍA), carrera, división y nacionalidad
    And solicita al alumno verificar que la información sea correcta

  Scenario: Visualización de la oferta de UEA
    Given que el alumno visualiza la sección "UEA Seleccionadas"
    Then el sistema muestra una tabla con las columnas: Seleccion, Clave, Nombre, Grupo, Horario y Profesor
    And el horario debe especificar los días y los intervalos de tiempo (ej. JUEVES: 09:30 AM - 11:00 AM)

  Scenario: Pre-selección automática de materias
    Given que el alumno carga la lista de materias
    When existen materias obligatorias o sugeridas según su avance académico
    Then el sistema las muestra seleccionadas automáticamente mediante un checkbox marcado
    And despliega el mensaje informativo: "Algunas UEA aparecen seleccionadas automáticamente, aun así puede editarlas."

  Scenario: Selección y deselección de materias
    Given que el alumno revisa la lista de UEA disponibles
    When el alumno marca el checkbox de una materia no seleccionada
    Or el alumno desmarca una materia previamente seleccionada
    Then el sistema actualiza visualmente la lista de materias que formarán parte de su inscripción

  Scenario: Visualización de detalles del profesor y grupo
    Given que el alumno consulta una materia específica (ej. 2156053 - Ingeniería de software I)
    Then el sistema muestra el código del grupo (ej. CP33)
    And el nombre completo del profesor asignado (ej. Humberto Gustavo Cervantes Macedo)
```
