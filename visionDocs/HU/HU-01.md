# Ingresar al sistema

## Tarjeta

**ID:** HU-01

**Nombre:** Ingreso al sistema SAPCyTI

**Como** usuario del sistema (administrador, profesor u otro rol autorizado)

**Quiero** ingresar al sistema mediante mi login y contraseña

**Para** acceder a la pantalla principal con las opciones correspondientes a mi tipo de usuario



##  Conversación

* El acceso al sistema se realiza desde la URL:
  `https://paoscalu.izt.uam.mx/sapcyti`
* Al ingresar, el navegador puede mostrar una advertencia relacionada con el certificado de seguridad del sitio.
* El usuario debe aceptar la advertencia del certificado para continuar.
* El sistema muestra una pantalla de autenticación con:

  * Campo **Email**
  * Campo **Contraseña**
  * Opción **No cerrar sesión**
  * Botón **ENVIAR**
  * Botón **LIMPIAR**
  * Liga de recuperación de contraseña
* Cuando el usuario ingresa credenciales válidas:

  * El sistema valida la información.
  * Si son correctas, muestra la pantalla principal.
  * Las opciones mostradas dependen del rol del usuario.
* Si las credenciales son incorrectas:

  * El sistema debe notificar el error.
  * No debe permitir el acceso.



##  Criterios de Aceptación 

```gherkin
Feature: Pantalla de ingreso al sistema SAPCyTI

  Scenario: Acceso exitoso al sistema
    Given que el usuario navega a la URL "https://paoscalu.izt.uam.mx/sapcyti"
    And acepta la advertencia del certificado del sitio
    When ingresa un email válido
    And ingresa una contraseña válida
    And presiona el botón "ENVIAR"
    Then el sistema valida las credenciales
    And muestra la pantalla principal
    And despliega las opciones de acuerdo al tipo de usuario

  Scenario: Acceso con credenciales incorrectas
    Given que el usuario se encuentra en la pantalla de login
    When ingresa un email o contraseña incorrectos
    And presiona el botón "ENVIAR"
    Then el sistema muestra un mensaje de error
    And no permite el acceso al sistema

  Scenario: Limpieza de campos
    Given que el usuario ha ingresado información en los campos de email y contraseña
    When presiona el botón "LIMPIAR"
    Then el sistema borra la información capturada en ambos campos

  Scenario: Recuperación de contraseña
    Given que el usuario se encuentra en la pantalla de login
    When selecciona la opción "¿Olvidaste tu contraseña?"
    Then el sistema redirige al proceso de recuperación de contraseña
```
