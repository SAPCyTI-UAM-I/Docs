# Cerrar sesión

## 1. Tarjeta
**ID:** HU-03

**Título:** Cerrar sesión

**Como:** usuario autenticado del sistema SAPCyTI

**Quiero:** cerrar mi sesión cuando termine de utilizar el sistema

**Para:** proteger mi información y evitar accesos no autorizados



## 2. Conversación
* Una vez que el usuario ha iniciado sesión correctamente, el sistema muestra la pantalla principal.
* En la parte superior o dentro del menú principal se encuentra la opción **“Cerrar sesión”**.
* Cuando el usuario selecciona dicha opción:

  * El sistema invalida la sesión activa.
  * Elimina la información de autenticación en curso.
  * Redirige automáticamente a la pantalla de login.
* Después de cerrar sesión:

  * El usuario no debe poder acceder a las opciones internas del sistema utilizando el botón “Atrás” del navegador.
  * Si intenta acceder a una URL interna sin autenticación, el sistema deberá redirigirlo al login.



## 3. Criterios de Aceptación
```gherkin
Feature: Cierre de sesión en SAPCyTI

  Scenario: Cierre de sesión exitoso
    Given que el usuario ha iniciado sesión correctamente
    And se encuentra en la pantalla principal del sistema
    When selecciona la opción "Cerrar sesión"
    Then el sistema invalida la sesión activa
    And redirige al usuario a la pantalla de login

  Scenario: Intento de acceso después de cerrar sesión
    Given que el usuario ha cerrado sesión correctamente
    When intenta acceder a una página interna del sistema mediante una URL directa
    Then el sistema redirige automáticamente a la pantalla de login
    And solicita autenticación

  Scenario: Uso del botón atrás después de cerrar sesión
    Given que el usuario ha cerrado sesión correctamente
    When presiona el botón "Atrás" del navegador
    Then el sistema no permite el acceso a la información protegida
    And redirige a la pantalla de login
```
